How to add to Gmail:
  1. Click the signature box below to select it, then press Ctrl+A (or Cmd+A on Mac) to select all
  2. Copy it (Ctrl+C / Cmd+C)
  3. In Gmail go to Settings → See all settings → Signature → Create new
  4. Paste into the signature box (Ctrl+V / Cmd+V)
  5. Save changes
